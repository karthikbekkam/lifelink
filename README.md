# 🩸 LifeLink - Smart Blood Donation Management System

## 📌 Overview
LifeLink is a Flask-based web application that connects blood donors with people in need. It provides secure authentication, donor search, blood request management, email notifications, and separate dashboards for Admin, Donor, and Needer.

## ✨ Features

- User Registration & Login
- Admin Dashboard
- Donor Dashboard
- Needer Dashboard
- Search Blood Donors
- Blood Request Management
- Forgot Password via Email
- Contact Form
- Google Maps Integration
- Profile Management

## 🛠️ Technologies Used

- Python
- Flask
- MySQL
- SQLAlchemy
- Flask-Login
- Flask-Mail
- Flask-Bcrypt
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Chart.js
